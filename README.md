# 🖥️ Windows 11 - Clone

A clean and modern Windows 11 desktop UI clone built using **HTML and CSS**.

This project recreates the basic look and feel of the Windows 11 desktop interface including,
- Task Bar
- Desktop Icons
- Weather Widget
- Search Bar
- System Tray

----

## 🚀 Features

- 🪟 Windows 11 inspired desktop UI
- 🔍 Search bar design
- 🌦️ Weather widegt
- 📶 System tray icons
- 🗂️ Desktop app icons
- ✨ Hover effects
- 🎨 Pure HTML & CSS only
- 📱 Clean layout and styling

---

## 🛠️ Technologies Used

- HTML
- CSS
